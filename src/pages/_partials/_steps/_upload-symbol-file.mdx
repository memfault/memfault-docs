At this point you should be able to generate a test events and crashes and push
the data to the Memfault UI.

You can confirm the error traces and crashes have arrived succesfully by
navigating to the "Issues" page. Follow the link pointed to below and upload a
symbol file.

![](/img/docs/mcu/first-trace.png)

After this step you will see the trace in the list of issues!

:::tip

You can programatically upload symbol files with the
[Memfault CLI tool](/docs/ci/install-memfault-cli).

:::
